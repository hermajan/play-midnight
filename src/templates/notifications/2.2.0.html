Hello once again, friends. Sorry it's been a while! Life got in the way with a new job and all that fun stuff, but I wanted to get an update pushed out to you guys with some fixes and a few new features.

FIRST NOTE: You may have seen a permission change on the extension requestion permissions for play-music.gstatic.com. This is where Google stores their static images, like their logo, which I needed access to to allow you to customize it. This was strictly for accessing images and nothing more.

I've had a few requests come in through Reddit for some new features, so I've added a couple of those, and some are still in the works. Always feel free to shoot me any other ideas/issues/suggestions you may have!

Now, on to the tweaks...

New Features

Dark Top Bar

By request I've added the ability to disable the accent color on the top search bar to allow you to keep it dark to match the rest of the theme.

Remove Radio Stations (EXPERIMENTAL)

This allows you to hide all those pesky radio stations on the 'Listen Now' page. However, since this is only experimental you'll see some overly large boxes/gaps in the recent activity and recommendations section.

Remove Songza

With this you're able to remove the Songza section (The pre-made playlists at the top of 'Listen Now').

Tweaks

Style Fixes

Updated the new Google logo to use your accent color. Fixed a few issues where styles were broken (such as dragging a song to playlist, etc).

Options Order

The options have been reorganized to hopefully make more sense. I've kept core features at the top, followed by accents, followed by all the other fun stuff (alphabetized).

Code Tweaks

I've made more changes to the core code, which shouldn't affect anyone other than people who poke around on Github.

Feedback

As always, I'm sure there will be bugs/issues. Please let me know, preferably via Reddit, email, or Chrome Webstore if you happen to encounter any. Screenshots are definitely the most helpful, or even css fixes if you're into that kind of thing. I'll get to these as quickly as I can.

Personal Message

If you'd like to help support this project in any way, please consider spreading the word via friends, Reddit, or by buying me some much needed coffee. Thanks again, Chris Tieman.
