Feature(s)

Larger Song Table

Due to a request I've added an option for Larger Song Table. This gives you the ability to switch back to the larger song rows instead of the tiny ones that Google introduced a few days ago.

There is a slight chance that the rating (thumb up/down) buttons might be broken, so if that's actually a thing, please let me know and I'll try and find a better solution.

Bugs

Queue Song Indicators

Fixed the issue with the song playing equalizer and play buttons being too big with the smaller song rows in the mini-queue.

Mini-Queue & Static Playlists Bug

Forgot to test the mini-queue when I was making the "Static Playlists" option and realized it was hidden. Quoth the Raven "Nevermore."

Recent Activity Background Bug

Fixed a bug I didn't account for on the Recent Activity when making the images larger. Had to do with user generated playlists not having a single image, but rather being multiple album covers.
