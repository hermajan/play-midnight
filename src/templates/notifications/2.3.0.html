Now, on to the tweaks...

New Features

Hide Menu Sections

By request I've added the ability to hide menu items from the sidebar (or hide it altogether if you feel saucy). You can do so by only checking the ones you'd like to see under the 'Visible Menus' section

Hide Auto-Playlists

By request I've added the ability to hide playlist items from the sidebar (or hide it altogether if you feel saucy). You can do so by only checking the ones you'd like to see under the 'Visible Auto-Playlists' section

Sound Search Auto-Playlist

Another requested auto-playlist has been added. This playlist is comprised of songs you've searched for via Google Now microphone search.

Tweaks

Style Fixes

Fixed an issue with the play queue going under the sidebar when 'Static Sidebar' was enabled on small screens. Fixed accent color on queue button.

Code Tweaks

I've made more changes to the core code, which shouldn't affect anyone other than people who poke around on Github.

Feedback

As always, I'm sure there will be bugs/issues. Please let me know, preferably via Reddit, email, or Chrome Webstore if you happen to encounter any. Screenshots are definitely the most helpful, or even css fixes if you're into that kind of thing. I'll get to these as quickly as I can.
