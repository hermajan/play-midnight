Features

Static Sidebar (EXPERIMENTAL)

Due to a request (and missing it myself) I've taken the time to add Static Sidebar back in. For those who have never seen/used it, this basically forces the left sidebar to always be open rather than hidden under a hamburger menu.

I had to rearrange a bunch of items, so there's a good chance something might break/not look right. Feel free to let me know and I'll try and fix them up.

Improvements

Recent Activity (YAYY!)

This was one that was really big to me. I'm not sure why Google decided that Recent Activity was such a secondary feature that they made it super tiny, but I've bumped those album arts up a bit to actually be useful.

Home Background Transition

Some of you may have noticed the flash of white on the Home page when scrolling from the first two items and down, this should (hopefully) no longer be an issue.

Larger Home Background Art

I actually really dig the new album art on the Home page, so I decided to bump up the album art a bit in size and I think it actually looks pretty fantastic. I hope you agree.

Bugs

Search Bar

Fixed text coloring issue on search suggestions.

Playlist Edit Dialog

Fixed up the accent colors on the edit playlist dialog.

Improve Suggestions/Header Jitter/Upload

Cleaned up the styling on the Improve Suggestions and Upload pages, fixed jitter bug on header when scrolling.

Miscellaneous

Fixed a few smaller styling issues I saw throughout. Dialogs for locations and such.

NOTE

I flew through a lot of these changes and tried a few new things I haven't done before. I haven't noticed any issues with performance, but if you find any issues, please let me know!

I had a ton of people message me about the black right click menu. I could never get the issue to reproduce for myself, but I've tried my best based on Github feedback to fix the issue. I'm hoping it's better for you, or that Google will push changes to everyone soon.
